# Pricing Data Sources

The Audit Engine logic relies on current retail pricing as of the submission week. All hardcoded logic and numbers trace back to these official pricing pages.

## ChatGPT (OpenAI)
*Source: https://openai.com/chatgpt/pricing*
- **Plus Plan:** $20 / user / month
- **Team Plan:** $30 / user / month (billed monthly) or $25 / user / month (billed annually). *Minimum 2 seats required.*
- **Enterprise Plan:** Contact Sales

## Claude (Anthropic)
*Source: https://anthropic.com/pricing*
- **Pro Plan:** $20 / user / month
- **Team Plan:** $30 / user / month. *Minimum 5 seats required.*

## Cursor (Anysphere)
*Source: https://cursor.com/pricing*
- **Pro Plan:** $20 / user / month
- **Business Plan:** $40 / user / month. *Adds centralized billing, SSO, and privacy mode enforcement.*

## Gemini (Google)
*Source: https://gemini.google.com/advanced*
- **Advanced Plan:** $20 / user / month (via Google One AI Premium).

## GitHub Copilot
*Source: https://github.com/features/copilot#pricing*
- **Individual:** $10 / user / month
- **Business:** $19 / user / month
- **Enterprise:** $39 / user / month

## Midjourney
*Source: https://docs.midjourney.com/docs/plans*
- **Basic:** $10 / month
- **Standard:** $30 / month
- **Pro:** $60 / month
